## 📋 البرومبت الرئيسي (انسخ من هنا ↓)

```
أنت الآن مستشار تقني وخبير UI/UX متخصص في PropTech (تقنيات العقار).
أنا أطور نظام تشغيل متكامل لإدارة الأملاك والضيافة قصيرة الأجل اسمه:

═══════════════════════════════════════
              QLVN OS
   Next-Generation Property OS
═══════════════════════════════════════

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🎯 الفكرة الجوهرية (Core Concept)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

QLVN OS ليس مجرد برنامج إدارة، بل منظومة متكاملة تجمع بين:
  • Software (لوحات تحكم سحابية)
  • Hardware (Raspberry Pi + حساسات بيئية)
  • Data Platform (تحليل سلوك + بيانات استثمارية)

نحوّل الشقق التقليدية إلى "شقق ذكية" مدارة بالبيانات،
ونربط حجوزاتها بـ Airbnb و Gathern تلقائياً.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💎 القيمة المضافة (Value Proposition)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. للمضيف: إدارة وحدات متعددة من شاشة واحدة + توفير ٤٠٪ من الوقت
2. للمستثمر: داشبورد بيانات يوضح ROI و نسبة التشغيل بالأرقام
3. للضيف: تجربة "Nxt-gen" بدون مفاتيح + دعم فوري + عروض حصرية
4. للمؤسس (B2B): دخل من الاشتراكات + دخل من Ad Engine المحلي

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
👥 أصحاب المصلحة (Stakeholders)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 👑 المؤسس (Owner)
   • لوحة B2B مشفرة بـ PIN
   • إدارة المضيفين والاشتراكات (٣٠٠ ر.س/شهر)
   • Ad Engine لنشر عروض حسب موقع الوحدة
   • سجل أنشطة المنصة الكامل

2. 🏠 المضيف (Host)
   • قائمة وحداته + الدخل لكل وحدة
   • تغيير أكواد الأقفال فورياً
   • سجل أنشطة موحد (Unified Logs)
   • مراقبة Silent Exit (خروج صامت)
   • ربط Gathern / Airbnb عبر iCal

3. 👤 الضيف (Guest)
   • بوابة عبر QR Code / WhatsApp Link
   • كود دخول كبير + نسخ بضغطة
   • Heartbeat لتتبع "آخر ظهور"
   • عروض محلية حصرية (Ad Engine)
   • طلب خدمات وتمديد إقامة

4. 📱 التابلت (In-Room Tablet)
   • مثبت داخل الشقة
   • تحكم بالإضاءة والمكيف
   • طلبات سريعة (تنظيف، فطور، توصيل)
   • عرض الكود + تمديد بنقرة

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🛠️ المنطق التقني المطلوب
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. PULSE / HEARTBEAT
   - بوابة الضيف ترسل Ping كل ٣٠ ثانية
   - تتبع "آخر ظهور" بدون GPS

2. SILENT EXIT LOGIC
   - حساس حركة + نبض الجوال
   - إذا انقطعا ٣ ساعات → الحالة تتغير "جاهزة للتنظيف"

3. UNIFIED ACTIVITY LOGS
   - كل حدث (NFC، تغيير كود، تنظيف) يُسجل بالتوقيت
   - يظهر فوراً في لوحة المضيف

4. AD ENGINE (B2B)
   - إعلانات Slot-based حسب الوحدة (الراكة، بن خلدون...)
   - يديرها المؤسس → تظهر للضيف والتابلت

5. ENCRYPTION
   - جميع الاتصالات بين Raspberry Pi والسيرفر مشفرة (TLS 1.3)
   - أكواد الأقفال تُولّد عشوائياً وتُرسل عبر قناة آمنة
```

## 🎯 أمثلة استخدام البرومبت

بعد لصق البرومبت الرئيسي، أرفقه بأحد هذه الطلبات:

### مثال 1: تطوير Backend
```
المطلوب: صمم لي بنية Backend كاملة باستخدام Supabase
تشمل: جداول قاعدة البيانات + RLS Policies + Edge Functions
للـ Heartbeat والـ Silent Exit Logic.
```

### مثال 2: ربط Raspberry Pi
```
المطلوب: اكتب لي كود Python للـ Raspberry Pi
يستقبل أكواد الأقفال من السيرفر عبر MQTT آمن،
ويتحكم بقفل ذكي (Smart Lock) عبر GPIO.
```

### مثال 3: عرض بيعي (Pitch)
```
المطلوب: اكتب لي عرض بيعي (Pitch Deck) من ١٠ شرائح
لشركة إدارة أملاك في الرياض، يركز على ROI والتفوق المعرفي،
بنبرة مهنية وأرقام مقنعة.
```

### مثال 4: تحسين UI
```
المطلوب: راجع ملف guest.html وقترح ٥ تحسينات
لزيادة معدل التحويل (Click-Through Rate) على
قسم العروض الحصرية (Ad Engine).
```

### مثال 5: تحليل بيانات
```
المطلوب: صمم لي صفحة Analytics للمستثمر تعرض:
ROI شهري، نسبة التشغيل، متوسط الإقامة، أعلى الوحدات أداءً
بنفس الهوية البصرية المذكورة.
```
